Global Attributes & Events

Global Attributes (work on most elements)

  • class - CSS class name(s)
  • id - Unique identifier
  • style - Inline CSS styles
  • title - Tooltip text
  • lang - Language of content
  • dir - Text direction (ltr, rtl)
  • hidden - Hide element
  • tabindex - Tab order
  • accesskey - Keyboard shortcut
  • contenteditable - Editable content
  • draggable - Drag and drop
  • spellcheck - Spell checking
  • translate - Translation hint
  • role - ARIA role
  • aria-* - ARIA attributes
  • data-* - Custom data attributes

Common Event Attributes

  • onclick - Mouse click
  • onload - Element loads
  • onsubmit - Form submission
  • onchange - Value changes
  • onfocus - Element gains focus
  • onblur - Element loses focus
  • onkeydown - Key pressed
  • onmouseover - Mouse enters
