Media Elements (Detailed Attributes)

<img>
Purpose: Embeds an image

Important Attributes:
  • src: URL/path to the image file.
  • alt: Descriptive text shown if image fails OR read by screen readers.
  • loading: "lazy" delays loading until itâ€™s near viewport; "eager" loads immediately.
  • decoding: "async" lets browser decode image off-main-thread; "sync" waits.
  • srcset & sizes: Provide responsive image versions based on viewport width.
  • usemap: Associates image with an <map> for clickable regions.
  • crossorigin: Controls fetching credentials; "anonymous" or "use-credentials".
Show Code
<audio>
Purpose: Embeds audio (music, podcasts, narration)

Important Attributes:
  • controls: If present, shows browser's default play/pause UI.
  • autoplay: Starts playback immediately; often muted to prevent intrusive audio.
  • loop: Repeats the audio continuously after ending.
  • muted: Silences audio; useful when combining with autoplay.
  • preload: Controls preload behavior: "none", "metadata", or "auto".
  • crossorigin: Controls credential behavior with cross-origin requests.
Show Code
<video>
Purpose: Embeds video content

Important Attributes:
  • controls, autoplay, loop, muted (same behavior as audio).
  • preload: Preload behaviorâ€”"metadata" recommended to avoid large downloads.
  • poster: URL to an image shown before playback starts.
  • playsinline: Prevents full-screen on mobile Safari; important for inline videos.
  • crossorigin: Credential mode for CORS video sources.
Show Code
<source>
Purpose: Provides multiple media variants (audio/video/image)

Important Attributes:
  • src: File source URL.
  • type: MIME type hint (e.g., video/mp4).
  • media: Media query, to select source based on screen condition.
  • srcset & sizes: For responsive `<picture>` images.
Show Code
<figure> / <figcaption>
Purpose: Group a media item and its caption

Important Attributes:
  • figcaption must be first or last child of `<figure>`.
Show Code
<picture>
Purpose: Provide responsive image sources

Important Attributes:
  • ` <source>` inside `<picture>` uses media/srcset.
  • `<img>` fallback uses standard `src`/`alt` attributes.

Global Attributes & Events

Global Attributes (work on most elements)

  • class - CSS class name(s)
  • id - Unique identifier
  • style - Inline CSS styles
  • title - Tooltip text
  • lang - Language of content
  • dir - Text direction (ltr, rtl)
  • hidden - Hide element
  • tabindex - Tab order
  • accesskey - Keyboard shortcut
  • contenteditable - Editable content
  • draggable - Drag and drop
  • spellcheck - Spell checking
  • translate - Translation hint
  • role - ARIA role
  • aria-* - ARIA attributes
  • data-* - Custom data attributes

Common Event Attributes

  • onclick - Mouse click
  • onload - Element loads
  • onsubmit - Form submission
  • onchange - Value changes
  • onfocus - Element gains focus
  • onblur - Element loses focus
  • onkeydown - Key pressed
  • onmouseover - Mouse enters
